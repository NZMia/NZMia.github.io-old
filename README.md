Personal Project Refactor
------
This is for `react` + `express` project
Local port; 9000

#### Quick start
--------

##### Install
```
  npm install
```
##### Dll
_The vendor needs to be packaged once when the first time running. Repackage it if dependencies change._
```
  npm run dll
```
##### Build
```
  npm run build
```
##### Start
```
  npm run Start
```
